# Version Polymorphic BatakJava

## System requirements

- Gradle (8.10.1)
- Apache Maven (3.9.6)
- > Java 8
- [Z3 Solver](https://github.com/Z3Prover/z3) version 4.8.10

### Gradle installation

- MacOS

Install using brew:

```
brew install gradle
```
